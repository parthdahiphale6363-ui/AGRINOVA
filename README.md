# 🌾 AGRINOVA - AI-Powered Farming Assistant

![Agrinova Banner](link-to-banner-image)

Created by **Parth Dahiphale**

## 🚀 Features
- 🤖 Multi-AI Crop Recommendation (5 ML Models)
- 🌤️ Live Weather with Location Detection
- 💰 Real-time Mandi Prices
- 🏛️ Government Schemes with Direct Apply Links
- 💬 AI Chatbot Assistant

## 🛠️ Tech Stack
- **Backend**: Python Flask
- **Frontend**: HTML5, CSS3, JavaScript
- **AI/ML**: scikit-learn, XGBoost, Ensemble Learning
- **APIs**: Open-Meteo Weather, Nominatim Geocoding
